# Entregas do Grupo 1

Esta pasta registra entregas concluídas pelo Grupo 1.

Cada entrega deve ter um arquivo Markdown próprio, usando o template:

```txt
templates/entrega-grupo-1.template.md
```

## Nome recomendado

```txt
g1-000-entrega-descricao-curta.md
```

Exemplo:

```txt
g1-001-entrega-inflacao-ine.md
```
