# Dados

Esta pasta contém os dados usados e produzidos pelo Grupo 1.

## Estrutura

```txt
data/
├── raw/
├── processed/
└── validated/
```

## Uso das pastas

| Pasta | Uso |
| --- | --- |
| `raw/` | Dados originais ou minimamente alterados. |
| `processed/` | Dados tratados, limpos e padronizados. |
| `validated/` | Dados revisados e prontos para Grupo 2 e Grupo 3. |

## Regra geral

Não mover um arquivo para `validated/` sem checklist de validação, fonte registrada e dicionário de dados atualizado quando aplicável.
