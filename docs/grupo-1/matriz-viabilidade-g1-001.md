# Matriz Preliminar de Viabilidade — G1-001

## 1. Objetivo

Registrar a viabilidade preliminar de fontes públicas para responder às perguntas de pesquisa enviadas pelo Grupo 3.

Esta matriz ainda não representa validação final de dados. Ela serve para indicar se existem fontes públicas suficientes, quais limitações foram encontradas e qual deve ser o próximo passo.

## 2. Prazo

- Entregas individuais: **26/05/2026 18:00 Europe/Lisbon**.
- Consolidação da matriz: **26/05/2026 23:59 Europe/Lisbon**.

## 3. Critérios de Viabilidade

| Status | Uso |
|---|---|
| Alta | Existe fonte pública acessível, com período e formato compatíveis. |
| Média | Existe fonte provável, mas com limitação de período, granularidade, formato ou metodologia. |
| Baixa | Existe apenas fonte indireta, agregada, desatualizada ou difícil de extrair. |
| Fonte não encontrada | Não foi identificada fonte pública suficiente para a pergunta. |

## 5. Observações de Consolidação

- Separar claramente IRS, contribuições à Segurança Social e prestações sociais.
- Registrar fonte parcial ou fonte limitada como entrega válida, desde que a limitação esteja explícita.
- Não alterar perguntas do Grupo 3 nesta matriz; perguntas inviáveis devem ser sinalizadas como risco.
- Após consolidação, atualizar a issue [#10](https://github.com/equipe-tratadados/tratamento-validacao-dados/issues/10) e reportar o resumo no grupo de líderes.
