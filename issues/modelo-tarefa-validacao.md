# Tarefa — Validação de Dataset

## 1. Solicitação

**Objetivo:** validar se um CSV está adequado ao padrão do Grupo 1 e pode ser usado pelos Grupos 2 e 3.

## 3. Critério de Aceite

A tarefa será considerada concluída quando houver:

- [ ] arquivo na pasta correta;
- [ ] nome seguindo o padrão definido;
- [ ] encoding UTF-8;
- [ ] separador `;`;
- [ ] cabeçalho presente;
- [ ] campos em `snake_case`;
- [ ] `fonte_id` correspondente no catálogo;
- [ ] dicionário de dados atualizado;
- [ ] limitações registradas;
- [ ] resultado final indicado: aprovado, aprovado com ressalvas ou rejeitado.

## 4. Resultado da Validação

```txt
Dataset: [nome]
Fonte ID: [fonte_id]
Status: [aprovado / aprovado com ressalvas / rejeitado]
Motivo: [resumo]
Próximo passo: [ação]
```
