# Scripts

Esta pasta deve conter scripts usados para coleta, limpeza, tratamento, validação ou automação.

## Regra

Todo script relevante deve estar ligado a uma task com ID.

Exemplo:

```txt
[G1-010] Criar script de validação de encoding dos CSVs
```

Branch:

```txt
g1-010/criar-script-validacao-encoding
```

Commit:

```txt
feat(validacao): adiciona script de validacao de encoding [G1-010]
```
